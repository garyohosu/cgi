さくら CGI ツールボックス

クイックチェック用の1ファイルAPI。

以下のカードを使用して、各CGIエンドポイントにアクセスしてください。 すべての呼び出しは、./now.cgi のように同一フォルダ内の相対パスで指定します。

GET ./now.cgi GET ./uuid.cgi POST ./validate.cgi GET ./convert.cgi
応答形式

成功: {"ok": true, "data": {...}}

失敗: {"ok": false, "error": {...}}

すべての応答は JSON で、キャッシュなしです。
